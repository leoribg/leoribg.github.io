Vantagens

A fase de prototipação no desenvolvimento de novos produtos é muito importante e serve como um meio para validar hipóteses de mercado.

O protótipo é uma forma de testar a viabilidade do produto antes de investir em um desenvolvimento mais complexo.

A prototipação é crucial para entender a complexa interação entre o usuário final e a solução.

Muitas vezes, um fator crucial para o acordo de uma oportunidade de negócio é o sucesso de uma prova de conceito (POC - Proof of Concept).

Uma POC é geralmente construida a partir de um lote piloto de protótipos. Os feedbacks dos clientes sobre a POC são essenciais para confirmar o valor das principais funcionalidades de um projeto.

Através de uma prova de conceito (POC - Proof of Concept) é possível testar a solução em clientes chaves e confirmar a necessidade real dos requisitos levantados.

Esse conjunto de funcionalidades é o que define o escopo do Produto Mínimo Viável (MVP - Minimum Viable Product).
